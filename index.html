Help & Guide
1
Upload your video
Drop or browse your finished reel (MP4 or MOV). This should be your final edited video before captions are added — after you export from CapCut, Premiere, or whatever editor you use.
  • File size: up to 2GB supported
  • Any resolution works — the tool detects height automatically
  • The video preview will appear once your file loads
2
Choose transcription engine and transcribe
Two options:
  • OpenAI API (Faster): Paste your API key. Costs ~$0.006 per minute — less than a cent per reel. Tap "How do I get an API key?" for a full guide.
  • Browser / Free: Uses Whisper running on your device. No cost ever. First use requires a one-time 74MB download. After that it's instant.
Optionally paste your voice script in the text box — this significantly improves punctuation and capitalization accuracy, especially for technical terms.
3
Choose caption style
Six styles to pick from. Amber Highlight is the default and performs best on most content. Adjust:
  • Words per line: 2 is punchy, 3 is standard, 4 suits longer sentences
  • Font size: 52px is the default. Increase for bigger screens or older viewers
  • Highlight color: Pick a preset or use the custom color picker
4
Edit captions
Every caption line is editable. Click any line to fix the text. Use the play button next to each line to scrub the video to that timestamp and check sync. Common edits:
  • Fix names or technical terms Whisper got wrong
  • Delete filler words or "um" lines
  • Split a long line into two shorter ones using Add Line
5
Position captions
Drag the caption up or down on the video preview to set its vertical position. Rules of thumb:
  • Stay above the red zone at the bottom — phones crop 20–30px off the bottom edge
  • For 9:16 reels, roughly 85–90% from the top works well for most content
  • Check that captions don't overlap your face or any key on-screen elements
6
Export and burn
On a computer:
  • Download captions.srt
  • Download the burn script for your OS (Mac or Windows)
  • Put both files in the same folder as your video
  • Double-click the burn script — it runs automatically
  • Upload output_captioned.mp4 to Instagram, Facebook, and YouTube Shorts — same file, no editing needed
On mobile: Download the SRT file and email it to yourself. Open this tool on your computer to complete the burn step.
The transcription doesn't match what I said
Paste your voice script into the script box in Step 2 — this is the single biggest accuracy improvement. Switch to the OpenAI API engine if you're using the browser engine, as the API version has higher accuracy on clear speech and technical vocabulary. For words that consistently come out wrong, just fix them manually in the editor in Step 4.
The burn script says "ffmpeg not found"
The burn script automatically downloads ffmpeg the first time it runs — but it needs an internet connection and the right system permissions.

Mac: If auto-download fails, open Terminal and run brew install ffmpeg. No Homebrew? Install it at brew.sh (one command, free).

Windows: Download ffmpeg from ffmpeg.org/download.html. Extract the zip, copy ffmpeg.exe into the same folder as your video and script, then re-run the script.
The model download failed or got stuck
Check your internet connection and try again using the Download button. The model is 74MB so it needs a stable connection. If it keeps failing, switch to the OpenAI API engine instead — it doesn't require the local model at all, and costs less than a cent per reel.
My captions are cut off at the bottom on phone
Go back to Step 5 (Position) and move the caption higher. The red zone at the bottom of the preview marks the unsafe area — keep captions above it. A Y position of roughly 85–90% from the top works on most phones. Phones vary in how much they crop — when in doubt, go higher.
The captions are out of sync with the audio
Use the play button next to each caption line in Step 4 to check sync. You can click the timestamp on any line and edit it manually. If everything is shifted by the same amount (e.g. all captions are 2 seconds late), it usually means the video had a silent intro — trim it before uploading, or shift all timestamps by opening the SRT file in a text editor and adjusting.
The video file is too large to upload
The tool supports up to 2GB which covers most reel exports. If your file is larger, compress it first using HandBrake (free, Mac and Windows). For a 60-second Instagram reel, 1080p at around 50MB is typical — if yours is much larger something went wrong in the export settings.
Do I need an internet connection to use the tool?
Once the Whisper model is downloaded, Steps 1, 3, 4, 5, and 6 work entirely offline. Step 2 (transcription) needs internet if you use the OpenAI API engine. The browser Whisper engine runs fully offline after the initial model download.
How do I remove the cached model to free up space?
Open Settings (gear icon, top right) → scroll to Whisper Model → click "Remove from device". This deletes the 74MB model from your browser cache. You can re-download it anytime from the same settings panel.
Is my API key safe?
Your API key is saved in your browser's local storage — it never leaves your device except when sent directly to OpenAI's servers for transcription. CaptionCraft has no backend server and never sees your key. To remove it from your browser, go to Settings → Clear all data.
